# Eclipse Common Security Infrastructure

Eclipse CSI develops technologies and practices to help secure the software supply chain of Eclipse Foundation projects.

This organization hosts security tooling, shared workflows, identity infrastructure, and supporting documentation for that work.

For project details, governance, and community resources, visit the [Eclipse CSI project page](https://projects.eclipse.org/projects/technology.csi).
